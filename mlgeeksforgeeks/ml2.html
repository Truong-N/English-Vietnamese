What is the matrix? Ma trận là gì?

A matrix is a collection of numbers arranged in a rectangular array in rows and columns. Ma trận là tập hợp các số được sắp xếp trong một mảng hình chữ nhật theo hàng và cột. In the fields of engineering, Trong các lĩnh vực kỹ thuật, physics, vật lý, statistics, thống kê, and graphics, và đồ họa, matrices are widely used to express picture rotations and other types of transformations. ma trận được sử dụng rộng rãi để biểu thị phép quay hình ảnh và các loại phép biến đổi khác.

The matrix is referred to as an m by n matrix, Ma trận được gọi là ma trận m x n, denoted by the symbol “m x n” if there are m rows and n columns. được biểu thị bằng ký hiệu "m x n" nếu có m hàng và n cột.

Mathematical Operations with Matrix in Python Các phép toán với Ma trận trong Python

Example 1: Ví dụ 1: Adding values to a matrix with a for loop in python Thêm giá trị vào ma trận bằng vòng lặp for trong python

Here, Ở đây, we are adding two matrices using the Python for-loop. chúng ta đang cộng hai ma trận bằng vòng lặp for của Python. 

# Program to add two matrices using nested loop
X = [[1, 2, 3],[4, 5, 6], [7, 8, 9]]
Y = [[9, 8, 7], [6, 5, 4], [3, 2, 1]]
result = [[0, 0, 0], [0, 0, 0], [0, 0, 0]]
# iterate through rows
for row in range(len(X)):
    # iterate through columns
    for column in range(len(X[0])):
        result[row][column] = X[row][column]+ Y[row][column]
for r in result:
    print(r)

Example 2: Ví dụ 2: Adding and subtracting values to a matrix with list comprehension Thêm và trừ các giá trị vào ma trận bằng cách hiểu danh sách

Performing the Basic addition and subtraction using list comprehension. Thực hiện phép cộng và trừ cơ bản bằng cách hiểu danh sách.

Add_result = [[X[row][column] + Y[row][column]
               for column in range(len(X[0]))] 
               for row in range(len(X))]
Sub_result = [[X[row][column] - Y[row][column]
               for column in range(len(X[0]))] 
               for row in range(len(X))]
print("Matrix Addition")
for r in Add_result:
    print(r)
print("\nMatrix Subtraction")
for r in Sub_result:
    print(r)

Slicing in Matrix using Numpy Cắt trong Ma trận bằng Numpy

Slicing is the process of choosing specific rows and columns from a matrix and then creating a new matrix by removing all of the non-selected elements. Cắt là quá trình chọn các hàng và cột cụ thể từ một ma trận, sau đó tạo một ma trận mới bằng cách xóa tất cả các các phần tử không được chọn. In the first example, Trong ví dụ đầu tiên, we are printing the whole matrix, chúng ta đang in toàn bộ ma trận, in the second we are passing 2 as an initial index, trong ví dụ thứ hai, chúng ta đang truyền 2 làm chỉ mục ban đầu, 3 as the last index, 3 làm chỉ mục cuối cùng, and index jump as 1. và chỉ mục nhảy là 1. The same is used in the next print we have just changed the index jump to 2. Tương tự như vậy được sử dụng trong lần in tiếp theo, chúng ta vừa thay đổi chỉ mục nhảy thành 2. 

X = np.array([[6, 8, 10], 
[ 9, -12, 15],
[ 12, 16, 20],
[ 15, -20, 25]])
# Example of slicing 
# Syntax: Lst[ Initial: End: IndexJump ]
print(X[:])
print("\nSlicing Third Row-Second Column: ", X[2:3,1])
print("\nSlicing Third Row-Third Column: ", X[2:3,2])
