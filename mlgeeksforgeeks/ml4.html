Add Two Matrices Using for loop Thêm hai ma trận bằng vòng lặp for

https://www.geeksforgeeks.org/python-program-add-two-matrices/
Here we will use for loop with two matrices for adding the elements. Ở đây chúng ta sẽ sử dụng vòng lặp for với hai ma trận để thêm các phần tử. 

# Program to add two matrices using nested loop
    X = [[1, 2, 3],
         [4, 5, 6],
         [7, 8, 9]]
    Y = [[9, 8, 7],
         [6, 5, 4],
         [3, 2, 1]]
    result = [[0, 0, 0],
              [0, 0, 0],
              [0, 0, 0]]
    # iterate through rows
    for i in range(len(X)):
        # iterate through columns
        for j in range(len(X[0])):
            result[i][j] = X[i][j] + Y[i][j]
    for r in result:
        print(r)

Add Two Matrices Using List Comprehension Thêm hai ma trận bằng cách sử dụng List Comprehension

In Python, Trong Python, we can implement a matrix as nested list (list inside a list). chúng ta có thể triển khai một ma trận dưới dạng danh sách lồng nhau (danh sách bên trong danh sách). We can treat each element as a row of the matrix. Chúng ta có thể coi mỗi phần tử như một hàng của ma trận. In this program we have used nested for loops to iterate through each row and each column. Trong chương trình này, chúng ta đã sử dụng các vòng lặp lồng nhau để lặp qua từng hàng và từng cột. At each point we add the corresponding elements in the two matrices and store it in the result. Tại mỗi điểm, chúng ta thêm các phần tử tương ứng vào hai ma trận và lưu trữ trong kết quả. 

# Program to add two matrices
    # using list comprehension
    X = [[1, 2, 3],
         [4, 5, 6],
         [7, 8, 9]]
    Y = [[9, 8, 7],
         [6, 5, 4],
         [3, 2, 1]]
    result = [[X[i][j] + Y[i][j] for j in range
               (len(X[0]))] for i in range(len(X))]
    for r in result:
        print(r)

Add Two Matrices Using zip() function Thêm hai ma trận bằng hàm zip()

The output of this program is the same as above. Đầu ra của chương trình này giống như trên. We have used nested list comprehension to iterate through each element in the matrix. Chúng tôi đã sử dụng sự hiểu biết danh sách lồng nhau để lặp qua từng phần tử trong ma trận. List comprehension allows us to write concise codes and we must try to use them frequently in Python. Sự hiểu biết danh sách cho phép chúng ta viết các mã ngắn gọn và chúng ta phải cố gắng sử dụng chúng thường xuyên trong Python. Here, Ở đây, The zip function accepts iterator i of each element(list) of matrix, Hàm zip chấp nhận iterator i của mỗi phần tử (danh sách) của ma trận, mapping them, ánh xạ chúng, adding them using sum(), thêm chúng bằng sum(), and storing them in the map form. và lưu trữ chúng ở dạng bản đồ. 

# Program to add two matrices using zip()
    X = [[1, 2, 3],
         [4, 5, 6],
         [7, 8, 9]]
    Y = [[9, 8, 7],
         [6, 5, 4],
         [3, 2, 1]]
    result = [map(sum, zip(*t)) for t in zip(X, Y)]
    print(result)

Add Two Matrices Using Numpy Thêm hai ma trận bằng Numpy

The numpy library has a built-in overload of the operator +, Thư viện numpy có hàm quá tải tích hợp của toán tử +, that allows one to perform the addition of matrices. cho phép thực hiện phép cộng các ma trận. 

import numpy as np
    X = [[1, 2, 3],
         [4, 5, 6],
         [7, 8, 9]]
    Y = [[9, 8, 7],
         [6, 5, 4],
         [3, 2, 1]]
    result = np.array(X) + np.array(Y)
    print(result)

Add Two Matrices Using SymPy Thêm hai ma trận bằng SymPy

To add matrices using a library, Để thêm ma trận bằng thư viện, you can use the Matrix class provided in the SymPy library. bạn có thể sử dụng lớp Matrix được cung cấp trong thư viện SymPy. 

from sympy import Matrix
    X = [[1, 2, 3],
         [4, 5, 6],
         [7, 8, 9]]
    Y = [[9, 8, 7],
         [6, 5, 4],
         [3, 2, 1]]
    # Create Matrix objects from the lists
    matrix_x = Matrix(X)
    matrix_y = Matrix(Y)
    # Add the matrices
    result = matrix_x + matrix_y
    # Print the result
    print(result)
